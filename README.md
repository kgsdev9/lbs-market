# LBS MARKET — Plateforme de vente de produit digital + de formation 

## 🚀 Démarrage rapide

```bash
npm install
npm run dev
```

Ouvre : **http://localhost:5173**

## 📄 Pages publiques

| URL | Description |
|-----|-------------|
| `/` | Page d'accueil (hero + formations + articles + projets) |
| `/formations` | Liste des formations avec filtres |
| `/formations/:slug` | Détail formation (programme, avis, inscription) |
| `/articles` | Liste des articles techniques |
| `/articles/:slug` | Article complet (prose markdown) |
| `/blog` | Blog KGS |
| `/blog/:slug` | Billet de blog |
| `/projets` | Projets open source |
| `/projets/:slug` | Détail projet (README, équipe, stats) |
| `/communaute` | Page communauté + membres |
| `/login` | Connexion |
| `/register` | Inscription |

## 🔧 Dashboard

| URL | Description |
|-----|-------------|
| `/dashboard` | Tableau de bord (KPIs, articles récents, activité) |
| `/dashboard/articles` | Liste articles |
| `/dashboard/articles/new` | Éditeur Markdown complet (toolbar, preview live) |
| `/dashboard/blog` | Liste blog |
| `/dashboard/formations` | Gestion formations |
| `/dashboard/formations/new` | Créer formation avec modules |
| `/dashboard/projets` | Gestion projets open source |
| `/dashboard/membres` | Gestion membres |
| `/dashboard/parametres` | Paramètres site + profil + sécurité |

## 🎨 Stack & Design

- **Vue 3** + Vite 5 + Pinia + Vue Router 4
- **Polices** : Instrument Sans + Space Grotesk + Geist Mono
- **Dark mode** natif via CSS variables
